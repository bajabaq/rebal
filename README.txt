###############################################################################

To Install:

Run z_install.bat

Which will give you the options to
*Copy over the files under revision control from the source media (USB stick)
to the c:\projects\rebal directory

*Install Python 3.8

*Install the various python packages required

*Install the rebal package and settings

-------------------------------------------------------------------------------

To Run:

From the shell:
Go to the bin directory (c:\projects\rebal\bin)
And run rebal.bat

Create 1 shortcuts on the desktop to launch the programs:
1. REBAL:
    python bin\rebal.bat
